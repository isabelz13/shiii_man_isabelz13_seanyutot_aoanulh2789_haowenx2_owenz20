NYC Voting & Income Explorer

About This Project

This project visualizes the relationship between political party regiestration and median household income across New York City.
This interactive map lets you explore how voting patterns and income levels are distrubted across the city; it uses real data from NYC election districts and ZIP code income reports.
Built by shiii_man®

How to Use the Map

  • The map displays NYC election districts color-coded by party registration concentration
  • Blue gradients represent Democratic registration, red gradients represent Republican registration
  • The income layer shows ZIP code regions shaded by median household income
  • Hover or click on a district to see its specific data

The lighter the color, the higher the income. The grayer the color, the less party stronghold there is in that district. Below is an example for a district that is majority democrat.

Low Income High Income
Dominant (80%): deep navy pale blue
Contested (51%): dark gray light gray

Data Sources

Party registration data sourced from the NYC Board of Elections.
Income data sourced from NYC Open Data / U.S. Census Bureau ZIP code income reports.
